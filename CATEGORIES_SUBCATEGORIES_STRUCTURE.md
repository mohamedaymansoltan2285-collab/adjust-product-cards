# نظام الأقسام والفئات الفرعية
## Categories & Sub-Categories System

---

## البنية الكاملة

### الأقسام الرئيسية (CATEGORIES) - 6 أقسام

1. **رجالي** (Men)
2. **حريمي** (Women)
3. **أطفال** (Kids)
4. **شبابي** (Youth)
5. **إكسسوارات** (Accessories)
6. **ملابس رسمية** (Formal Wear)

### الفئات الفرعية (SUB_CATEGORIES) - 25 فئة

قمصان، بناطيل، فساتين، جاكيتات، سويترات، تيشيرتات، جينز، بدل، معاطف، شورتات، تنانير، ملابس داخلية، أحذية، حقائب، ساعات، أحزمة، أوشحة، قبعات، نظارات شمسية، مجوهرات، عطور، ملابس رياضية نشطة، ملابس السباحة، ملابس الاسترخاء، أطقم

---

## المسارات والصفحات

### 1. الصفحة الرئيسية (`/`)
- **CategoriesSection**: عرض الأقسام الرئيسية الـ 6
- **SubCategoriesCarousel**: عرض الفئات الفرعية بشكل دوائر جميلة مع دوران تلقائي كل 6 ثوان

### 2. صفحة القسم الرئيسي (`/category/[id]`)
- عرض جميع المنتجات للقسم الرئيسي المختار
- خيارات الترتيب (الأحدث، السعر، الشهرة)
- تحميل Skeleton أثناء جلب البيانات

### 3. صفحة الفئة الفرعية (`/subcategory/[id]`)
- عرض جميع المنتجات للفئة الفرعية المختارة
- نفس خيارات الترتيب
- نفس تصميم تحميل البيانات

---

## إضافة منتج جديد

عند إضافة منتج في `/dashboard/add-product`:

1. **اختر القسم الرئيسي** من قائمة `CATEGORIES`
2. **اختر الفئة الفرعية** من قائمة `SUB_CATEGORIES`
3. ملء باقي البيانات

\`\`\`javascript
// في form data:
{
  category: "men",           // من CATEGORIES
  subCategory: "shirts",     // من SUB_CATEGORIES
  // ... باقي البيانات
}
\`\`\`

## تصميم الكاروسيل (SubCategoriesCarousel)

### المميزات:
- ✅ دوائر جميلة بألوان متنوعة (10 ألوان)
- ✅ عرض 5 فئات في كل مرة
- ✅ دوران تلقائي كل 6 ثواني
- ✅ نقاط تنقل للانتقال بين الشاشات
- ✅ إيقاف تلقائي عند النقر على نقطة التنقل
- ✅ تأثيرات Hover جميلة (تكبير + ظل)

### الألوان المستخدمة:
\`\`\`
1. Blue → Cyan
2. Purple → Pink
3. Amber → Orange
4. Emerald → Teal
5. Rose → Red
6. Indigo → Purple
7. Green → Emerald
8. Yellow → Amber
9. Cyan → Blue
10. Pink → Rose
\`\`\`
